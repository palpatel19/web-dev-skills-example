WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text MarkupLanguage. HTML is the content of a web page. Examples of HTML includes headings, paragraphs, images, links, lists etc.

Check out the HTML Tutorial on w3schools.com.

Learn CSS

CSS stands for Cascading Style Sheet. CSS is the style of a web page. Examples of CSS include font styles and colour, background colour, borders, dimensions, layout,etc.

Check out the CSS Tutorial on w3schools.com.

Learn Javascript

JavaScript is a programming language of the web. Javascript helps to make web pages interactive and dynamic. Some common JavaScript tasks are:

  • add new HTML to the page,
  • change the existing content / style
  • react to user events (mouse clicks, key presses,etc.)

Check out the Javascript Tutorial javascript.info.
